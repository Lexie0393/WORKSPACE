3 - Selectores

Selectores de tipo:

Se refieren a elementos HTML específicos. Por ejemplo, el selector p apuntará a todos los párrafos en tu documento HTML.

Selectores de clase:

Utilizan el atributo class para aplicar estilos a uno o varios elementos. Se escriben con un punto seguido del nombre de la clase. Por ejemplo, .boton apuntará a todos los elementos con class="boton".

Selectores de ID:

Utilizan el atributo id para apuntar a un elemento HTML específico. Se escriben con un símbolo de numeral (#) seguido del nombre del ID. Por ejemplo, #encabezado apuntará a un elemento con id="encabezado".

Selectores de atributo:

Permiten seleccionar elementos basados en sus atributos y valores. Por ejemplo, [type="text"] apuntará a todos los elementos <input> con type="text".

Selectores descendientes:

Permiten apuntar a un elemento dentro de otro. Por ejemplo, ul li apuntará a todos los elementos <li> que estén dentro de una lista <ul>.

Selectores hijos directos:

Seleccionan elementos que son hijos directos de otro elemento. Por ejemplo, ul > li apuntará a todos los elementos <li> que sean hijos directos de una lista <ul>.

Selectores combinados:

Permiten combinar múltiples selectores para ser más específicos en la selección de elementos. Por ejemplo, header nav ul seleccionará todas las listas <ul> que estén dentro de un elemento <nav> que a su vez esté dentro de un <header>.

Selector de adyacente (+):

Selecciona el elemento inmediatamente después de otro elemento específico. Por ejemplo, h2 + p seleccionará el primer párrafo que esté directamente después de un elemento <h2>.

Selector de hermano general (~):

Selecciona todos los elementos que están después del elemento especificado, aunque no sean adyacentes. Por ejemplo, h2 ~ p seleccionará todos los párrafos que estén después de cualquier <h2>.

Selector de primer hijo (:first-child):

Selecciona el primer hijo de un elemento padre específico. Por ejemplo, p:first-child seleccionará todos los párrafos que sean el primer hijo de su elemento padre.

Selector de último hijo (:last-child):

Selecciona el último hijo de un elemento padre específico. Por ejemplo, p:last-child seleccionará todos los párrafos que sean el último hijo de su elemento padre.

Selector de enésimo hijo (:nth-child):

Permite seleccionar elementos basados en su posición dentro de un elemento padre. Por ejemplo, p:nth-child(2) seleccionará todos los párrafos que sean el segundo hijo de su elemento padre.

Selectores de pseudo-clase:

Representan estados especiales de elementos. Por ejemplo, :hover aplica estilos cuando el cursor está sobre un elemento, :first-child selecciona el primer hijo de un elemento, etc.

Selectores de pseudo-elementos:

Permiten estilizar partes específicas de un elemento. Por ejemplo, ::selection que sirve para darle estilo al texto que seleccionas con el ratón o ::first-letter que aplica estilos a la primera letra de la primera línea.
