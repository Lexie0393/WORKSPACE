Ejercicios de funciones en JavaScript

Primera fase

Ej.1 - Volumen del cubo

Vamos a crear una función llamada volumenCubo() que le pida al usuario que introduzca la longitud del cubo. Una vez lo introduzca, vamos a mostrar el resultado en una alerta.

Ej.2 - Volumen de una esfera

Vamos a crear una función llamada volumenEsfera() que le pida al usuario que introduzca el radio de una esfera. Una vez lo introduzca, vamos a mostrar el resultado en una alerta.

Ej.3 - Volumen de un cilindro

Vamos a crear una función llamada volumenCilindro() que le pida al usuario que introduzca el radio de la base y la altura de un cilindro. Una vez lo introduzca, vamos a mostrar el resultado en una alerta.

Ej.4 - Volumen de un objeto tridimensional

Ahora vamos a pedir al usuario que nos introduzca el objeto a la que queremos calcular su volumen (CIL, ESF o CUB). Una vez escogida el objeto, que le solicita los datos necesarios para calcular el volumen de esta. Al final mostrar en una alerta el volumen del objeto indicado.

Segunda fase

Ej.1 - Mostrar mensaje

Vamos a crear una función llamada mostrarMensaje() que le pida al usuario que introduzca un texto. Una vez lo introduzca, vamos a escribir ese texto en el elemento párrafo que tiene el id="ejercicio-1".

Resultado:

Este texto va a ser sustituído por lo que introduzca el usuario.

Ej.2 - Color card

Vamos a cambiar el color de la card de este ejercicio. Para ello necesitamos crear una función cambiarColor() en la que le preguntemos al usuario un color, y le establezcamos ese color a esta card.
Pista: una vez pedimos el color, debemos obtener en una variable el elemento de la card (con getElementById, QuerySelector) y a la propiedad style y background le asignamos el valor de ese color.

Ej.3 - Color fondo

Ahora vamos a cambiar el color de fondo de toda la página por el color que nos inserte el usuario. Esta vez tienes que darle nombre a la función, asociarla al botón y conseguir obtener el elemento que equivale a toda la página y cambiarle el color de fondo.

Ej.4 - Texto del revés

Pedir al usuario que escriba un texto cualquiera y devolverlo al revés. Devuelve el resultado en el párrafo con id="ejercicio-4"

Resultado:

Este texto va a ser sustituído por lo que introduzca el usuario pero al revés.
